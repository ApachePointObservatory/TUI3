TUI:Building TUI from Source On Windows

This file is for people who wish to build TUI for Windows from source.

Warning: this is an obscure document and some instructions may be out of date.

Install the Python and the support packages listed under unix installation, but with these refinements:

  • You will also need to install pywin32.
  • Most of the packages come with binary installers. For those that don't:
    • Unpack the directory
    • Use Command Line to cd to the directory (type cd and drag the folder)
    • Type "setup.py install". Note: no "python"! Also, if you forget the ".py" you will get a dialog box with a strange error.
    • pyfits may fail to install unless you have the right version of Visual Studio. You can work around this by editing setup.py and commenting out the line of the setup call that mentions the extension (as of this writing the only C extension is optional and not required by TUI).

To build the TUI Windows Application.

  • Open the terminal
  • cd to the BuildForWindows directory
  • Type "setup.py py2exe"
  • At the end you should have directory TUI_<vers>_Windows. You may zip this up and distribute it.

Note: I don't update my Windows installation very often and when I do it usually breaks the setup.py file. So you may need to do some tweaking. Please let me know of any problems and fixes. -- Russell
